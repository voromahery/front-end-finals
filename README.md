## How To Use

We'll be using `node-sass` for our Sass compilation

- `npm i`

To compile the **Sass**, run the command below:

- `npm start`

```bash
# Clone this repository
$ git clone https://github.com/your-user-name/your-project-name

# Install dependencies
$ npm install

# Run the app
$ npm start
```
